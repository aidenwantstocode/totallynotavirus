Curious of the potential childhood nostalgia stored within the drive, you plug it into an old family computer you haven't used since high-school, only to find out that the little flash drive isn't as harmless as you thought it was."

***TotallyNotAVirus*** is an interactive, OS Simulator style survival game heavily inspired by *KinitoPET* and *Welcome To The Game*. You will try to survive against certain corruptions in an old Windows 98-style system, trying to preserve the hidden encrypted data inside of the retaliating flash drive. 

The game isn't projected to have too many scary elements but this might change in the upcoming future based on changes in creative direction and technical decisions. Although, the core gameplay dynamics will still feature fast-paced periodically initiated waves of threats as well as active player defense mechanisms of the game environment. The game is currently being actively developed on C++ using SFML. Be sure to check this repository once in a while if you're interested!

### 🎭 Promised Features (The Player's Experience)
* **True-to-Life Simulation:** A fully stable sandbox mimicking a late-90s operating system complete with responsive button clicks, smooth desktop navigation, and standard desktop apps.
* **The "Infection" Trigger Event:** A seamless, unprompted transition from a safe simulation environment into an active psychological horror scenario triggered by an in-game script (e.g., opening a rogue email attachment).
* **Smart Malware Behavior:** Rather than static pre-rendered jumpscares, the virtual virus interacts dynamically with the player's actions—forcefully spawning popups, shifting windows away from the mouse cursor, and messing with application text inputs.
* **Hacker Survival Mechanics:** Players must use the built-in simulated Command Line Interface (CLI) to find hidden directory pathways, analyze log files, and execute decryption scripts to "cleanse" the system before the virus triggers a total virtual crash.

More features coming very soon!
